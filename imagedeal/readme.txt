环境说明：
    Python 3.8
    opencv-python 4.11.12
    pyqt5 3.15.14

文件说明：
    image_deal:
    用于存放采样裁剪的样本图片
    source_imagedata:
    用于存放原始水果图片

    baidufruit.py:对接百度水果识别API
    fruit_recongnition.ui:原始qt资源文件
    fruit_recongnition.py：qt生成的Python代码
    switch.py:用于水果阈值设置和部分判断
    main_image_process:主要处理函数
    text.py和text2.py：前期功能模块测试

运行方式：
    主函数在main_image_process.py，点击该文件运行
